- Windows (any)

### How to compile

Just type in your shell:

~~~bash

# With CMake as build system

libelf> mkdir build
libelf> cd build
build> cmake ..
build> make

# With Makefile as builder

libelf> make
~~~

### How to install

~~~bash
build> sudo make install
~~~
